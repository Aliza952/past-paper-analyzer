# 📊 Past Paper Analyzer

## 🚀 Problem

Students often struggle to analyze past exam papers and identify the most important topics. This leads to inefficient preparation and wasted time.

---

## 💡 Solution

**Past Paper Analyzer** is an AI-powered system that automatically extracts key topics from exam papers and generates:

* 📊 Topic analytics
* 📅 Smart study plans
* 📚 Practice suggestions
* 📘 Syllabus coverage insights

---

## 🔥 Features

* 📄 Upload PDF exam papers
* 🤖 AI-based topic extraction (Ollama / llama3)
* 📊 Topic frequency visualization (Bar Chart)
* 📈 Topic distribution (Pie/Donut Chart)
* ⭐ Ranked important topics with difficulty levels
* 📅 Auto-generated study plan (day-wise)
* 🤖 AI insights for quick preparation
* 📘 Syllabus coverage analysis (covered vs missing topics)
* 📚 Practice suggestions

---

## ⚙️ How to Run Locally

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Aliza952/past-paper-analyzer.git
cd past-paper-analyzer
```

### 2️⃣ Backend Setup

```bash
cd backend
npm install
node server.js

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```
